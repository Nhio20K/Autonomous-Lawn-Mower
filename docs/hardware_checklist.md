# 🚜 Master Checklist: การนำหุ่นยนต์ลงสนามจริง (Hardware Deployment)

รายการตรวจสอบนี้รวมทั้งขั้นตอนหน้างานและ **"จุดที่ต้องแก้ในโค้ด"** ทั้งหมดไว้ในที่เดียวครับ

---

## 🟢 1. จุดที่ต้องแก้ไขในโค้ด (Configuration Changes)
**ทำสิ่งนี้ก่อนเริ่มรันระบบบนหุ่นยนต์คันจริง:**

- [ ] **[nav2_params.yaml] เปลี่ยน Plugin เลี้ยว**:
  - หาบรรทัด `plugin:` ในส่วน `FollowPath`
  - เปลี่ยนจาก `RegulatedPurePursuitController` เป็น `RotationShimController`
- [ ] **[ekf_sim.yaml] เปลี่ยนแหล่งพิกัด**:
  - หาบรรทัด `odom1:` ในส่วน `ekf_filter_node_map`
  - เปลี่ยนจาก `/ground_truth_odom` เป็น `/odometry/gps`
- [ ] **[start_robot.sh] เปลี่ยนชื่อไฟล์รั้ว**:
  - แก้ `--geofence_file` จาก `lawn_geofence_sim.yaml` เป็น `lawn_geofence.yaml`
- [ ] **[hardware_bringup.launch.py] เช็คปลั๊ก USB**:
  - ตรวจสอบพอร์ต `/dev/ttyUSBX` และ Baudrate ของ Lidar และ RTK ตัวใหม่
- [ ] **[mow_zigzag.py] เปิดโหมด Manual Go**:
  - บรรทัด 213: แก้ให้เป็น `cmd = input(...)` (ต้องลบ `'go' #` ออก) เพื่อให้หุ่นหยุดรอคำสั่ง

---

## 🟢 2. การติดตั้งหน้างาน (Physical Setup)
- [ ] **Port Check**: รัน `ls /dev/ttyUSB*` เพื่อเช็คพอร์ต USB ที่เสียบจริง
- [ ] **Master Base Station**: ตั้ง Base ในที่โล่ง และเช็คสัญญาณส่งไปหารถ (Rover)
- [ ] **ตรวจสอบ GPS**: ต้องขึ้นสถานะ **"RTK Fixed"** เท่านั้นก่อนเริ่มงาน

---

## 🟢 3. การสร้างรั้วและแผนที่ทำงาน (Teach-in Phase)
- [ ] **ถอดใบมีด**: เพื่อความปลอดภัยในการทดสอบรั้วรอบแรก
- [ ] **รัน `./start_robot.sh`**: และเลือก **โหมด 2 (Teach-in)** ในแท็บ GEOFENCE
- [ ] **บันทึกรั้วจริง**: ขับหุ่นวนรอบสนาม กดบันทึกจุด และกด `s` เพื่อคัดลอกไฟล์รั้วจริง
- [ ] **เช็ค Rviz**: เส้นสีแดง (รั้ว) และเส้นสีม่วง (ทางตัด) ต้องทับกันและอยู่ในพื้นที่สนามจริง

---

## 🟢 4. การเริ่มภารกิจ (Starting Mission)
- [ ] **เช็คทางเดิน**: ดูว่าไม่มีวัตถุขวางทางที่หุ่นจะแล่นไปจุดเริ่ม
- [ ] **สั่งลุย**: ไปที่แท็บ MOWING พิมพ์ `go` แล้วคอยสังเกตการณ์
- [ ] **มืออยู่ที่ปุ่มหยุด**: พร้อมกด Stop หรือ `Ctrl+C` ทันทีหากหุ่นวิ่งผิดปกติ

---
> [!IMPORTANT]
> **หมายเหตุ**: หากมีการแก้ไขไฟล์ `.py` อย่าลืมรัน `colcon build --packages-select robot_bridge` เพื่อให้อัปเดตค่าล่าสุดครับ!
